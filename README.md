# raw-lang
=========

raw-lang runtime and compiler package.

List of 100% supported systems:
- NeedleSystem (https://github.com/dataph-labs/needlesystem-src)
- NetBSD
- Linux

Build and Install.
------------------

NeedleSystem:
    
    Built into the base system

NetBSD:

    make
    su
    make install
    exit

Linux:

    make -f Makefile.linux
    sudo make install -f Makefile.linux
