# raw-lang
=========

raw-lang runtime and compiler package.

Build and Install.
------------------

Linux:

    make -f Makefile.linux
    sudo make install -f Makefile.linux

QUICK START
-----------

Compile example code:
    ./rawcc example/hello_world.rawc rbuild/resourses.rawb

Try run:
    ./raw-runtime rbuild/resourses.rawb
